# 🚀 AgentScope-PaaS 快速启动指南

## 一键启动项目

### Windows 用户
双击以下文件即可运行：
- `run_all.bat` - 启动前后端服务
- `run_backend.bat` - 仅启动后端
- `run_frontend.bat` - 仅启动前端

### Linux/Mac 用户
```bash
# 一键启动全部服务
./run_all.sh

# 或单独启动
./run_backend.sh    # 后端
./run_frontend.sh   # 前端
```

## 📍 访问地址

启动后访问：
- **前端**: http://localhost:3000
- **后端API**: http://localhost:8000/api/v1/docs

查看 [RUN_GUIDE.md](RUN_GUIDE.md) 了解更多详细信息。

## ⚡ 系统要求

- **后端**: Python 3.8+
- **前端**: Node.js 16+

## 🔧 首次使用

1. 确保已安装Python和Node.js
2. 双击运行启动脚本
3. 等待依赖自动安装
4. 在浏览器中访问上述地址
