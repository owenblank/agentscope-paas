## ✨ 新增功能

### 1. 技能上传组件
**位置**: `frontend/src/components/Agent/SkillsUploadForm.tsx`

#### 支持三种上传方式
- 📄 **单个文件**: 上传SKILL.md文件（最大3MB）
- 📁 **文件夹**: 上传包含技能文件的文件夹（最大3MB，最多20个文件）
- 📦 **ZIP压缩包**: 上传ZIP格式的技能包（最大3MB，最多50个文件）

#### 主要特性
- 拖拽上传支持
- 实时上传进度显示
- 文件格式和大小自动验证
- 已上传技能列表管理
- 符合skill-creator规范

## 🚀 使用方法

### 技能上传步骤

1. **选择上传方式**
   - 在技能配置面板中选择上传方式
   - 支持单个文件、文件夹、ZIP压缩包

2. **准备SKILL.md文件**
   ```markdown
   ---
   name: my-skill
   description: 我的技能描述，说明何时触发和功能
   ---

   # 技能内容
   技能的具体指令和说明...
   ```

3. **上传文件**
   - 点击或拖拽文件到上传区域
   - 系统自动验证文件格式和大小
   - 等待上传完成

4. **配置执行参数**
   - 设置最大并发技能数
   - 配置超时时间和重试策略
   - 设置权限和安全级别

### 知识库配置步骤

1. **配置平台知识库**
   - 输入平台知识库服务URL
   - 选择认证方式（Bearer Token/API Key/OAuth2）
   - 设置认证令牌

2. **配置检索参数**
   - 选择检索模式（语义/关键词/混合）
   - 设置相似度阈值
   - 配置返回结果数量

3. **设置权限控制**
   - 配置每会话最大查询次数
   - 选择允许的操作类型
   - 设置访问权限

## 📋 配置规范

### SKILL.md文件规范
- **必需字段**: name, description
- **name格式**: 小写字母、数字、连字符
- **description长度**: 50-500字符
- **文件格式**: UTF-8编码的Markdown文件
- **文件大小**: 最大3MB

### 上传限制
- 单个文件: 最大3MB
- 总上传大小: 最大10MB
- 文件数量: 最多20个（文件夹模式）
- ZIP内文件数: 最多50个
