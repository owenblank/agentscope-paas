## 🔧 配置文件优化建议

### 1. 添加前端元数据
```yaml
# 在现有配置基础上添加
frontend_config:
  # 表单显示配置
  display_config:
    # 字段显示名称
    display_names:
      agent_metadata: "基础信息"
      model_config: "模型配置"
      prompt_config: "提示词配置"
      tool_config: "工具能力"

    # 字段显示顺序
    field_order:
      - agent_metadata
      - model_config
      - prompt_config
      - tool_config
      - memory_config
      - behavior_config

    # 条件显示规则
    conditional_display:
      - condition: "agent_type == 'FunctionCallAgent'"
        show_fields: ["tool_config"]
      - condition: "agent_type == 'DialogAgent'"
        hide_fields: ["tool_config"]

  # 表单验证规则
  validation_rules:
    agent_id:
      pattern: "^[a-z0-9_]+$"
      min_length: 3
      max_length: 50
      required: true

    api_key:
      pattern: "^sk-[a-zA-Z0-9-]{20,}$"
      required: true

    temperature:
      min: 0.0
      max: 2.0
      default: 0.7

  # UI组件类型
  ui_components:
    agent_type:
      type: "radio"
      options:
        - value: "DialogAgent"
          label: "对话智能体"
          description: "适合基础对话场景"
        - value: "FunctionCallAgent"
          label: "函数调用智能体"
          description: "支持工具调用"

    temperature:
      type: "slider"
      min: 0.0
      max: 2.0
      step: 0.1
      show_value: true

    system_prompt:
      type: "textarea"
      rows: 10
      placeholder: "输入智能体的角色设定和行为准则..."
      word_count: true
      quality_score: true
```

### 2. 配置模板定义

template_config:
  template_id: "customer_service_basic"
