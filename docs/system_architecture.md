## 🚀 部署架构

### 开发环境
```yaml
version: '3.8'
services:
  frontend:
    build: ./frontend
    ports: ["3000:3000"]
    environment:
      - VITE_API_URL=http://localhost:8000

  backend:
    build: ./backend
    ports: ["8000:8000"]
    environment:
      - DATABASE_URL=postgresql://user:pass@db:5432/agentscope
      - REDIS_URL=redis://redis:6379
    depends_on: [db, redis]

  db:
    image: postgres:14
    environment:
      - POSTGRES_DB=agentscope
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=pass
    volumes:
      - postgres_data:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    ports: ["6379:6379"]

  agent_runtime:
    build: ./runtime
    environment:
      - API_URL=http://backend:8000
    volumes:
      - ./configs:/app/configs
      - ./logs:/app/logs

volumes:
  postgres_data:
```


## 🔒 安全架构

### 认证授权流程
```
┌─────────────┐     1. 登录请求      ┌─────────────┐
│    用户     │ ──────────────────▶ │   API网关    │
└─────────────┘                     └─────────────┘
     ▲                                    │
     │ 4. 返回JWT Token                   ▼
     │                            ┌─────────────┐
     │                            │  认证服务    │
     │                            └─────────────┘
     │                                    ▼
     │                            ┌─────────────┐
     │                            │ PostgreSQL  │
     │                            └─────────────┘
```

### 数据安全
- **传输加密**: TLS 1.3
- **存储加密**: AES-256
- **API密钥**: 哈希存储 (SHA-256)
- **敏感数据**: 字段级加密
- **审计日志**: 完整操作记录
